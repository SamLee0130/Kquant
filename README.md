# Kquant

ETF 포트폴리오 백테스트 대시보드

**데모**: https://kquant-6ms3ipabequx4ghncwmufd.streamlit.app/

- **자산 배분 백테스트**: ETF 포트폴리오의 과거 성과 시뮬레이션 (리밸런싱, 인출, 배당금 포함)
- **한국 세금 반영**: 배당소득세(15%), 양도소득세(22%) 자동 계산 및 이연 납부 시뮬레이션
- **포트폴리오 비교**: 최대 5개 자산 배분 전략 간 성과 비교 분석
